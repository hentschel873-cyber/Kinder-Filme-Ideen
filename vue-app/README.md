# Vue + TypeScript Beispiel (Vite)

Dieses Beispiel zeigt, wie man eine kleine Vue 3 + TypeScript Anwendung mit Vite startet.

Schnellstart:

```powershell
cd vue-app
npm install
npm run dev
```

Öffne `http://localhost:5173` im Browser.

Hinweis: Dieses Beispiel ist bewusst minimal gehalten und läuft unabhängig vom Hauptprojekt.
