# Kinder-Filme-Ideen

Kleines statisches Frontend-Projekt mit wenigen JS- und CSS-Dateien.

Quickstart

1. Lokalen statischen Server starten (keine Installation erforderlich):

```powershell
npm run start
```

2. Optional: moderner Dev-Flow mit `vite` (wenn du es installieren willst):

```powershell
# einmalig
npm install --save-dev vite
# dann
npm run dev
```

Lint & Tests

- Lint: `npm run lint` (installiere `eslint` als dev-dependency)
- Tests: `npm test` (installiere `jest` als dev-dependency)

Projektstruktur (relevant)

- `index.html` — Einstiegspunkt (wurde hinzugefügt)
- `main.js` — App-Wiring / Event-Bindings
- `counter.js` — Zähler-Logik
- `style.css` — Styles
- `.github/copilot-instructions.md` — Hinweise für KI-Agenten

Wenn du möchtest, dass ich `vite`, `eslint` und `jest` automatisch installiere und konfiguriere, sag Bescheid.
